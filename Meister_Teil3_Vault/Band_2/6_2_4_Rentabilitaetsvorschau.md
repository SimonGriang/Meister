# 6.2.4 Rentabilitätsvorschau und Kapitaldienstfähigkeit

Die Rentabilitätsvorschau (Gewinnplan) stellt die erwarteten Umsätze den erwarteten Kosten gegenüber, um die langfristige wirtschaftliche Tragfähigkeit des Vorhabens zu beurteilen.

## Grundlagen der Planung
- **Zeitraum:** Erstellung für mehrere Jahre (Berücksichtigung von Tilgungsbeginn nach Anlaufphasen).
- **Zweck:** Vorlage bei Banken, Grundlage für Gutachten, Einschätzung der Erfolgsaussichten.

## Schema der Rentabilitätsvorschau
| Position | Berechnungsschritt |
| :--- | :--- |
| **Umsatz** | |
| - Materialverbrauch / Wareneinsatz | |
| **= Rohgewinn I** | |
| - [[6_2_2_Kostenplan|Personalkosten]] | |
| **= Rohgewinn II** | |
| + Sonstige betriebliche Erträge | |
| - [[6_2_2_Kostenplan|Sachkosten]] (Miete, Energie, Kfz, etc.) | |
| **= Erweiterter Cashflow** | |
| - Zinsen | |
| **= Cashflow** | |
| - Abschreibungen | |
| **= Gewinn / Betriebsergebnis** | **(Endpunkt der Rentabilitätsvorschau)** |

## Ermittlung der Kapitaldienstfähigkeit
Zusätzlich zur Rentabilität muss geprüft werden, ob der Betrieb seine Zahlungsverpflichtungen (Kapitaldienst) dauerhaft erfüllen kann.

**Schema zur Deckungsprüfung:**
  Gewinn / Betriebsergebnis
+ Abschreibungen (da nicht zahlungswirksam)
- Tilgung betrieblicher Darlehen
- [[6_1_3_e_Privatentnahmen|Saldo aus privaten Ausgaben und Einnahmen]]
---
**= Überdeckung (+) / Unterdeckung (-)**

## Bedeutung für den Gründer
Die Qualität dieser Planung entscheidet über die Bewilligung von Finanzierungen und Fördermitteln. Eine umfassende Beratung durch HWK-Betriebsberater oder Steuerberater wird dringend empfohlen.
